# 📱 My Apps — Android Downloads

Free Android builds of my apps. Both are also live on the **iOS App Store**.
No Play Store needed — just download and install.

> ### 📲 How to install on Android
> 1. Tap a **Download** button below to get the `.apk` file.
> 2. Open the downloaded file. Android will warn about installing from an unknown source → tap **Allow from this source**.
> 3. Tap **Install**, then open the app. Done.
